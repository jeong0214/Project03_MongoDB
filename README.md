-웹에서 MongoDB에 저장된 작성내용을 수정페이지를 통해 수정가능
-swiper를 활용해 active된 슬라이드 이미지 크기 변경
-작성페이지에서 input요소를 제대로 채워야 데이터를 전송가능하게 미 작성시 경고문 등장
